Codurance CSS

Codurance CSS is a lightweight library built on top of Bulma
This library was built to standardise the look and feel of the websites created by Codurance.

How to change things

The library is build with node.js and npm, so you are going to need both before starting. Bulma has a quite easy way of overriding the main styles when using `sass`, we just have to set the following variables:

  • $primary
  • $info
  • $success
  • $warning
  • $danger
  • $dark
  • $family-primary

We do this in a `scss` file that will be processed and will create a `css` file. We can configure which files are going to be set in the `package.json` file. The `css-build` command takes the `codurance.scss` file and process to `dist/codurance.css`, so if a new file has to be added it has to be added to the processes just change the `css-build` step. 

                    
{
    "scripts": {
        "css-build": "node-sass --omit-source-map-url sass/codurance.scss dist/codurance.css",
        "css-watch": "npm run css-build -- --watch",
        "start": "npm run css-watch"
    }
}

Changing Bulma styles

To change styles in Bulma you can override the variables for the library, you can see them here When changing the `codurance.scss` the `@import` for bulma always has to be in the bottom so it can read the variables declared. 

                
@charset "utf-8";
@import url('https://fonts.googleapis.com/css?family=Open+Sans');

// Codurance Colors
$codurance-red: #d60d74;
$codurance-orange: #ef7726;
$codurance-yellow: #ffd000;
$codurance-purple: #5c64b9;
$codurance-green: #185466;
$codurance-light-green: #51bfbb;
$coudrance-font: 'Open Sans', Arial, sans-serif;

    
// Update Bulma Global variables
$primary: $codurance-orange;
$info: $codurance-purple;
$success: $codurance-light-green;
$warning: $codurance-yellow;
$danger: $codurance-red;
$dark: $codurance-green;
$family-primary: $coudrance-font;

// Import bulma.sass is always after declaring the variables
@import "../node_modules/bulma/bulma.sass";
